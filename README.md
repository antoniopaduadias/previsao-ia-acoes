# Previsão de Tendências de Ações com Machine Learning

Este projeto automatiza a extração de dados históricos da Bolsa de Valores (B3), calcula indicadores técnicos nativos de volatilidade e treina um modelo de inteligência artificial para prever tendências de curtíssimo prazo para o ativo PETR4 (Petrobras).

## Tecnologias Utilizadas
- Python 3.14
- Scikit-Learn
- Pandas
- Matplotlib

## Como Executar
1. Instale as dependências básicas:
pip install yfinance pandas scikit-learn matplotlib --break-system-packages

2. Execute o script principal:
python graficos_acoes.py
